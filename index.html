# MeMes' World - Meme Drop Website A simple, clean website where strangers can drop memes without logging in. Once submitted, memes cannot be deleted. ## Features - No login required - Permanent meme submissions (cannot be deleted) - Clean, modern design - Responsive layout - Category filtering - Anonymous or named submissions ## How to Use 1. Open `index.html` in your browser 2. Fill in the meme details: - Title - Image URL (direct link) - Your name (optional) - Category 3. Click "Submit Meme" 4. View memes in the gallery section ## Setup for GitHub Pages 1. Create a new GitHub repository 2. Upload all files to the repository 3. Go to Settings â†’ Pages 4. Set source to "main branch" 5. Your site will be live at: `https://[username].github.io/[repository-name]` ## Important Notes - Memes are stored in browser's localStorage (for demo purposes) - For production, you would need a backend server with a proper database - Image URLs must be direct links to images ## Technologies Used - HTML5 - CSS3 (with Flexbox and Grid) - JavaScript (Vanilla) - Font Awesome icons - Unsplash images for samples ## License MIT License - Feel free to use and modify!
